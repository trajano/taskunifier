Calendar



1. Calendar view
There are two type of calendars, you can select them in the right panel.

You can add new events in these calendars.
You can also move and resize events.

Double click on an event to edit the task.
Press ALT + C to complete/uncomplete tasks.
2. Select calendar mode
Select the calendar mode:
- Day view
- Week view
3. Select the calendars to display
Calendar "Tasks by due date":
Each event in this calendar represent a task with a due date.
The event start date is the due date of the task minus its length.
The event end date is the due date of the task.
If the length is less than 30mn, a length of 30mn is used to display the event.

If you do not use due time (use due time setting is disabled), the day end setting will be used as due time.

The icon is if the task is completed.
The icon is if the task is not overdue.
The icon is if the task is overdue.
Calendar "Tasks by start date":
Each event in this calendar represent a task with a start date.
The event start date is the start date of the task.
The event end date is the start date of the task plus its length.
If the length is less than 30mn, a length of 30mn is used to display the event.

If you do not use start time (use start time setting is disabled), the day start setting will be used as start time.

The icon is if the task is completed.
The icon is if the task is not completed.
4. Select the date to display
Select the day/week you want to display in the calendar view.
