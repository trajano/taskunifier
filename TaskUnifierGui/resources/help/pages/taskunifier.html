Welcome to TaskUnifier Help

TaskUnifier is a task management software based on the well-known GTD (Getting things done) methodology.
It will help you manage your tasks and therefore your time.

TaskUnifier is free and open source.
Only the synchronization with third-party websites requires plugins which are independent from TaskUnifier's project.
The plugins can be free or not.

Things to know

1) Task Searcher

Via the "+" button on the bottom left of the screen, you can create your own filters.
The filters allows you to display only some tasks and also allows you to sort them. For more information click here.

2) Task Template

Via the menu "Tasks -> Manage task templates", you can create and edit your own templates.
A task template allows you to create task with predefines values. For more information click here.

3) Synchronization

Via "Settings -> Synchronization", you can select a synchronization plugin which will allow you to synchronize your data with a third-party website or application like Toodledo.

It is a two-way synchronization, which means your data will be the same in TaskUnifier or in the third-party website or application.

Only one synchronization plugin can be enabled.


4) Publication

Via "Settings -> Publication", you can select a publication plugin which will allow you to publish your data to a third-party website or application like Google Calendar.

It is a one-way publication, which means your data from TaskUnifier will be sent to the third-party website or application but if you change something in the third-party website or application, these changes won't be replicated in TaskUnifier.

You can enable as many publication plugins you want.
